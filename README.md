# Бот по автопродаже звёзд на фанпее

## Как работает бот:
Бот постоянно мониторит FunPay и если кто то покупает звёзды, то бот через ваш телеграм аккаунт отправляет звёзды по юзернейму, который вписал покупатель. Никакие лишние сообщения нельзя писать ни продавцу, ни покупателю, это нужно будет написать в описании. Также вам нужно обязательно написать автоответ.

Для начала вам нужно установить язык программирования python и vscode, а также настроить их. В инете много хороших видосов про это

## Настройка бота - достаточно объёмная. Первым делом нам нужно получить все нужные нам переменные и вписать их в файл config.py.
Переменные:
1. GOLDEN_KEY - заходим на свой аккаунт в фанпей через ПК и нажимаем f12, выбираем вкладку Application, там будет вкладка coockies. Нажав туда, выпадет окошко funpay. Выбираем его и в окошке справа ищем переменную golden_key, копируем его значение и вставляем его в файл config.py
2. gifts_id - уже написан
3. Чтобы получить API_ID и API_HASH, вам нужно зайти на сайт my.telegram.org и авторизоваться с аккаунта телеграм, с которого будут отправляться подарки. Далее нужно нажать кнопку "API development tools". После вам нужно создать приложение, чтобы бот смог заходить к вам на аккаунт и отправлять подарки. Вы можете вписать данные только в поля "App title", "Short name", "Platform", а остальное можете не заполнять. После создания, вам нужно скопировать api_id и api_hash и внести эти переменные в файл config.py.
4. Ещё нам нужен обычный телеграм бот, который будет нам в случае чего отправлять уведомления. Для этого мы заходим в телеграм, заходим в бота @BotFather, запускаем его. Дальше вводим команду /newbot и делаем всё по инструкции, которую предоставляет бот. В конце он выдаст токен, который нужно будет записать в поле TELEGRAM_BOT_TOKEN, который находится в файле config.py. И теперь вам нужно запустить созданного вами бота с того аккаунта, с которого вы хотите получать уведомления(Нажать старт, он ответа не выдаст)
5. USER_TELEGRAM_ID - в пункте 4, вы должны были запустить бота, а теперь нам нужен id этого аккаунта, чтобы на него пресылать уведомления. Чтобы получить id, вы можете запустить бота @userinfobot, там выдастся id, вы его копируете и вставляете в config.py
